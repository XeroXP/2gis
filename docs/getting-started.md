# Установка проекта

**Установка**

1.Скачать проект
```
php
git clone git@github.com:Maksclub/2gis.git maksclub
cd maksclub
composer update
```
2.Переименовать файл /config/_db.php в db.php и настроить свою БД

3.Применить миграции:
```php
./yii migrate/up
```
